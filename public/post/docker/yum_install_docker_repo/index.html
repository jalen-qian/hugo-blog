【docker】在Centos8.0下用yum安装Docker报找不到源解决办法

在阿里云服务器（Centos8系统）安装doker时，报找不到源，如下图：

解决

方案：使用 Docker 仓库进行安装

设置仓库

安装需要的软件包，这里安装了yum-utils，yum-utils 提供了 yum-config-manager ，并且 device mapper 存储驱动程序需要 device-mapper-persistent-data 和 lvm2

1
2
3

sudo yum install -y yum-utils \
  device-mapper-persistent-data \
  lvm2

配置使用国内阿里云的源地址

1
2
3

sudo yum-config-manager \
    --add-repo \
    http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

安装docker-ce

1

sudo yum install docker-ce docker-ce-cli containerd.io

ps:如安装失败请戳这里
