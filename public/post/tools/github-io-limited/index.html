近期发现收藏的github.io页面都访问不了了，刚开始还以为是被墙了，后来才发现可以通过网络设置解决。

废话不多说，直接上解决方案：

解决方案

1.打开 控制面板\网络和 Internet\网络和共享中心

2.点击“属性”

3.双击 Ipv4

4.输入Dns服务器

先选择使用下面的DNS服务器地址 然后输入 223.5.5.5.5

5.打开网页查看能否访问
